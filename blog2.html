Cloud Computing




Cloud Computing is a model of service provisioning where shared resources, software and information are provided over the Internet to computers and other devices on demand. Cloud computing provides computation, file storage, networking, and other services that do not require end-user knowledge of the physical location and configuration of the system that delivers them. These systems reside over the network and share resources globally to deliver applications and data. The term cloud computing is often used interchangeably with Software as a Service (SaaS), Platform as a Service (PaaS) and Infrastructure as a Service (IaaS). Cloud Computing is a model of service delivery for enabling ubiquitous, convenient, on-demand network access to a shared pool of configurable computing resources (e.g., networks, servers, storage, applications, and services) that can be rapidly provisioned and released with minimal management effort or service provider interaction. Cloud Computing provides solutions for many different industries, including financial services, banking, insurance, healthcare, telecommunications, retail, education, government, and others.
